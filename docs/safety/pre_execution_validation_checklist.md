# 预执行校验清单

## 目标
在执行前完成语法、风险、效果三重校验，防止误执行。

## 三步校验
1. 语法校验
- 命令是否可被目标 Shell 解析。
- 关键参数是否缺失。

2. 风险评估
- 是否涉及删除、覆盖、权限修改、服务变更。
- 影响范围是单文件、目录、系统级还是网络级。

3. 效果模拟
- 用 echo、find、grep 先展示候选对象。
- 输出拟执行数量与目标路径。

## 必填确认信息
- 操作对象范围。
- 预估影响数量。
- 回滚方式。
- 执行窗口（是否业务高峰）。

## 通过条件
- 语法通过。
- 风险等级低于阈值，或已完成审批。
- 用户确认与审批记录一致。

## 拒绝条件
- 语法错误。
- 命令命中黑名单。
- 影响范围超出策略边界。
