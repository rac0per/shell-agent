\chapter{总结与展望}

\section{工作总结}

本文围绕面向中文Shell智能体的领域RAG增强系统展开研究，主要工作如下：

\begin{enumerate}
    \item 构建了包含38篇文档、覆盖150余种用户意图的Shell领域知识库，并提出系统性文档增强策略，将模板查询集MRR从0.92提升至1.0；
    \item 提出混合检索+重排策略，结合关键词路由过滤，在80个自然语言查询评估集上达到Recall@1=0.85、MRR=0.91；
    \item 设计并实现三级命令安全执行框架，含权限分级、预执行三步流水线和执行日志系统，通过58个单元/集成测试用例验证；
    \item 构造包含80个自然语言查询的评估集，设计2×2消融实验，系统分析了各模块的独立贡献与交互效果。
\end{enumerate}

\section{未来展望}

\begin{enumerate}
    \item \textbf{学习型路由}：用轻量分类器（TF-IDF+SVM或fastText）替换关键词规则，解决跨类硬例的路由错误问题；
    \item \textbf{双类别召回}：对跨类查询同时召回两个类别的候选集再融合排序；
    \item \textbf{同义词词典扩充}：系统性收集失败查询的同义词，补充至对应文档的同义问法章节；
    \item \textbf{重排超参数调优}：将关键词权重系数0.2的人工调参替换为在评估集上的自动优化；
    \item \textbf{在线学习}：记录用户采纳的文档，持续更新向量索引权重。
\end{enumerate}
