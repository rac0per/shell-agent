## 标准操作步骤
1. 查看目录所有文件（含隐藏）：`ls -alh <目录>`
2. 按修改时间降序排列：`ls -lt <目录>`
3. 按文件大小降序排列：`ls -lS <目录>`
4. 递归列出子目录：`ls -lhR <目录>`

## 推荐命令
ls -alh
ls -lt
ls -lS
ls -lhR

## 参数说明
- `-a`：显示隐藏文件（以 `.` 开头）。
- `-l`：长格式，含权限、属主、大小、修改时间。
- `-h`：文件大小使用人类可读格式（K/M/G）。
- `-t`：按修改时间降序排序。
- `-S`：按文件大小降序排序。
- `-R`：递归列出所有子目录。

## 风险与边界
- 只读查询命令，无写操作风险。
- 大目录递归列出可能输出量大，建议配合 `| head` 截断。

## 回滚方案
- 查询类命令无需回滚。

## 验收标准
- 命令正常输出文件列表，无报错。
- 排序结果与预期方向一致（时间或大小）。
