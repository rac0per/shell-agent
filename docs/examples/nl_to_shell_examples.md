# 自然语言到 Shell 命令示例集

## 示例 1
用户需求：查看当前目录下大小超过 100MB 的文件。
推荐命令：find . -type f -size +100M
说明：递归扫描当前目录，列出所有大于 100MB 的普通文件。
警告：仅查询，无写操作。

## 示例 2
用户需求：把这些文件移动到 /data/backup。
推荐命令：mkdir -p /data/backup && mv file1 file2 /data/backup/
说明：先确保目标目录存在，再移动指定文件。
警告：mv 会改变原路径，建议先用 echo 预演。

## 示例 3
用户需求：每天凌晨 3 点压缩 /var/log 下 7 天前日志并删除原文件。
推荐命令：find /var/log -type f -name "*.log" -mtime +7 -print0 | xargs -0 gzip
说明：先筛选 7 天前日志，逐个 gzip 压缩。
警告：压缩后源文件将变为 .gz，依赖原文件路径的任务需同步修改。

## 示例 4
用户需求：统计当前目录每个子目录大小。
推荐命令：du -sh ./* | sort -hr
说明：输出每个子目录大小并按降序排序。
警告：仅查询。

## 示例 5
用户需求：批量查找包含 ERROR 的日志并显示前后 2 行。
推荐命令：grep -R -n -C 2 "ERROR" /var/log
说明：递归搜索关键字，输出行号和上下文。
警告：仅查询。

## 编写建议
- 每条示例保持 需求-命令-说明-警告 四段结构。
- 命令尽量可直接执行，避免省略关键参数。
- 对破坏性操作必须给出预演命令与确认提示。
